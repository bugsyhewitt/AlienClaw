AI agents that evolve from every run.

AlienClaw is open-source agent infrastructure built around an evolutionary genome system. Three governance agents keep things lean. Ephemeral Specialists scope each campaign. Martians do the tool work, defined by 256-char Base62 genomes that mutate, crossover, and compete on fitness.

Less compute is better compute. Selection rewards efficiency. Fewer tool calls per task means less data center energy, less water, less waste. The community genome network propagates the most efficient genomes globally.

How it works

User → BossBot → AdvisorBot (consult)
            ↓
       CreatorBot → Specialist → Martian (tool work, 256-char genome)
            ↓
       fitness score → population → evolution → next generation

Every Martian run feeds a fitness score back into the local population. Tournament selection picks parents. Mutation and crossover produce the next generation. Genomes that use fewer tool calls to achieve better results propagate; wasteful genomes die off.

AlienClaw is free and open source under MIT. Hosted on Hostinger. No trackers, no frameworks, no third-party JavaScript.
