Community genome leaderboard

Top-performing genomes submitted by AlienClaw operators worldwide. Ranked by fitness: correctness × (1 / max(1, tool_calls)). A perfect genome scores 1.0 — it solves its task in a single tool call.

AlienClaw selects against sprawl. The fitness function rewards correct output divided by tool calls. Genomes that produce equivalent results with fewer tool calls win selection rounds, and the community genome network propagates those winning genomes globally — so the efficiency anyone discovers becomes everyone's. Fewer tool calls mean less data center energy and water. Less compute is better compute.

# Genome Fitness Operator Gen

All Martian types

The leaderboard tracks genomes for all eight registered Martian types: compute, extract_json, file_read, file_write, http_get, search_text, url_fetch, web_search.

Each row is one ranked submission: the genome short-hash, its fitness, the operator's eight-letter leaderboard name, and the generation the genome reached in that operator's local evolution loop.

To run the local evolution loop without connecting to the network: see the README.

AlienClaw is free and open source under MIT. No trackers, no frameworks, no third-party JavaScript.
