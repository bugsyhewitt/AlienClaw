All Martian types

The leaderboard tracks genomes for all eight registered Martian types: compute, extract_json, file_read, file_write, http_get, search_text, url_fetch, web_search.

Fitness is computed as correctness × (1 / max(1, tool_calls)). A genome that solves a task in one tool call with perfect correctness scores 1.0. The leaderboard shows the highest-fitness genomes observed across all operator installs that participate in the network.

To run the local evolution loop without connecting to the network: see the README.

AlienClaw is free and open source under MIT. No trackers, no frameworks, no third-party JavaScript.
