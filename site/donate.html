Support AlienClaw

AlienClaw is free and open source. It always will be.

What donations fund

Running api.alienclaw.net — the community genome network server — costs money. Server costs, storage, bandwidth. Bugsy's runway is limited. If AlienClaw is useful to you, consider supporting it.

  • Server hosting for api.alienclaw.net
  • Domain and infrastructure costs
  • Time to maintain and extend the project

Donations go directly to Bugsy, the project's primary maintainer. There's no foundation, no company, no investors. This is an indie open-source project.

GitHub Sponsors

The easiest way to donate is through GitHub Sponsors. One-time or recurring. GitHub matches contributions for eligible maintainers.

Other ways to support

  • Star the repo: github.com/AlienTool/AlienClaw — visibility helps.
  • Participate in the genome network: run AlienClaw, submit your high-fitness genomes, help the community pool grow. Compute is the contribution.
  • Open issues and PRs: bug reports, test improvements, tool runner enhancements. The project follows an identical-in-both-languages discipline — cross-language contributions especially welcome.
  • Tell people: the environmental thesis only holds at scale. More operators running AlienClaw means a stronger global selection signal.

What I won't promise

Donations don't buy roadmap priority, private access, or special treatment. AlienClaw is open source and the roadmap is public. Contributions are welcome from everyone regardless of whether they've donated.

If you donate, thank you. If you can't, no pressure — running the local evolution loop is contribution enough.

AlienClaw is free and open source under MIT. No trackers, no frameworks, no third-party JavaScript.
