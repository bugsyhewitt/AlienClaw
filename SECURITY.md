# Security Policy

## Reporting a Vulnerability

To report a security vulnerability in AlienClaw, please email:
**security@v3x.tech**

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge receipt within 48 hours and provide a resolution timeline.
Do not disclose publicly until we have had reasonable time to address the issue.

## Scope
This policy covers the AlienClaw repository and associated tooling at github.com/bugsyhewitt/AlienClaw.
